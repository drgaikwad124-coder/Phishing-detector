About Phishing Detection System

A machine learning based web application designed to detect phishing websites and help users identify unsafe URLs before sharing sensitive information.

What is Phishing?

Phishing is a type of cybercrime in which attackers create fake websites or messages that appear to be legitimate. These deceptive techniques are used to trick users into revealing sensitive information such as usernames, passwords, and financial details.

Phishing attacks often rely on social engineering rather than technical vulnerabilities, making them difficult to detect through traditional security mechanisms.

How This System Works

  • User submits a URL for analysis
  • Important URL and content-based features are extracted
  • The trained model evaluates the features
  • The website is classified as Safe or Phishing

Technologies Used

  • Backend: Python (Flask Framework)
  • Machine Learning: Scikit-learn
  • Frontend: HTML, CSS, Bootstrap
  • Libraries: NumPy, Pandas, BeautifulSoup
